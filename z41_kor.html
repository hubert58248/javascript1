Pętle w JavaScript

Pętla for

Konstrukcja:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

Opis: Wykonuje się określoną liczbę razy. Składa się z trzech części: początek, warunek, krok.

Pętla while

Konstrukcja:

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Opis: Sprawdza warunek przed każdą iteracją. Może się nie wykonać ani razu.

Pętla do...while

Konstrukcja:

let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

Opis: Najpierw wykonuje kod, potem sprawdza warunek. Zawsze wykona się przynajmniej raz.
